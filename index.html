What is Google Dorking?

Google dorking is a technique to search for sensitive information using advanced search operators. Using this tool you can generate queries and use those queries to find information with finer control.

Filters

Filter Description Example
allintext Searches for occurrences of all the keywords given. allintext:"keyword"
intext Searches for the occurrences of keywords all at once or one at a time. intext:"keyword"
allinurl Searches for a URL matching all the keywords in the query. allinurl:"keyword"
inurl Searches for a URL matching one of the keywords. inurl:"keyword"
allintitle Searches for occurrences of keywords all at a time. allintitle:"keyword"
intitle Searches for occurrences of keywords in title all or one. intitle:"keywords"
allinanchor Shows sites which have keyterms in links pointing to them, in order of the most links. Using all keywords. allinanchor:"keyword"
inanchor Shows sites which have keyterms all at a time in links pointing to them, in order of the most links. Using one or all keywords. inanchor:"keyword"
site Specifically searches that particular site and lists all the results for that site. site:"reddit.com"
filetype Searches for a particular filetype. filetype:"pdf"
link Searches for external links to pages. link:"keyword"
related List web pages that are similar to a specific web page. related:"nytimes.com"
cache Shows the version of a web page that Google has in its cache. cache:"linkedin.com"

Operators

Symbol Name Description Example
* Wildcard Used as a placeholder for any word or character. site:linkedin.com/in/*
+ Include Forces the inclusion of a search term. +intext:cats
- Exclude Forces the exclusion of a search term. -site:indeed.com
& AND Used in between terms, results must include both. site:linkedin.com & site:glassdoor.com
| OR Used in between terms, results may or may not include both. site:nytimes.com | site:washingtonpost.com
~ Synonyms Used to find search results with that are similar to a term, like synonyms. ~set
